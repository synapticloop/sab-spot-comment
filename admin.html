sab-spot-comment

A totally free auto commenter for SabNZBD/Newznab/SpotNAB applications

Administration options

Although there are a quite a few options, the only ones that you need to change are:

  • Server mode,
  • SabNzbd server,
  • Newznab server

Server mode

There are only two options here - either LIVE or DEMO.

LIVE mode will start posting comments immediately, DEMO mode will show you what comments would have been created.

SabNzbd server

[SABNZB URL] Enter the URL to your sabnzbd server, this should be in the form of 

http://your.server.name:9095

[SABNZB API KEY] Enter you sabnzbd api key, this can be found at 

http://your.server.name:9095/sabnzbd/config/general/

In the section 'API Key'. Note that you may have to click on the button 'Generate New Key' if one oesn't exist.

  • sabnzbd installation, and
  • newznab host

Newznab server

[NEWZNAB URL] Enter the url to your newznab based server, this should be of the form 

https://news.server.name.com/

[NEWZNAB API KEY] Enter your newznab api key. Go to your user account and it will be listed under Site Api/Rss Key

That's it

Click on the submit button at the bottom of the page - and you should be ready to go.

If you want to tweak more options - see details on the more admin page.
